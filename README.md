### What is Jangle?
Jangle is a content management system built on __NodeJS__, __MongoDB__, and __Elm__.

### Local Development

1. Install [Docker](https://docs.docker.com/engine/installation/)

1. Run __`docker-compose up`__

1. Change code, change things!
